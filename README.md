# AdventOfCode
This repo contains my attemps are various years of [adventofcode](https://adventofcode.com) challenges


The naming scheme for files is as follows:
* **dayXpuzX** - files contain source code for the puzzle 1/2 of a day. The extension can vary depending of the language used
* **dayXinp.txt** - files contain the puzzle input for that day
* **dayXinp_smpl** - files contain the sample input for the puzzle
